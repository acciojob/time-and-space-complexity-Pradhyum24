Time Complexity and Space Complexity

The time complexity of an algorithm quantifies the amount of time taken by an algorithm to run as a function of the length of the input. Note that the time to run is a function of the length of the input and not the actual execution time of the machine on which the algorithm is running on.efficent

Space Complexity refers to the amount of memory an algorithm uses while executing. It describes how the memory requirement grows with input size, and it is often expressed using Big-O notation such as O(n).
